 Sales Analysis and Forecasting Project

1 Objective

To contribute to business success by applying data analysis techniques and time series forecasting to generate insights and predict future sales performance.

2 Tools Used

MS Excel
Power BI

3 Project Description

* Developed an interactive dashboard with key KPIs and filters for detailed data exploration
* Performed data analysis to evaluate sales performance using visualizations
* Applied time series analysis to forecast sales for the next 15 days
* Generated actionable insights to support business decision-making

4 Data Cleaning

* Removed unnecessary columns
* Checked and corrected data types
* Identified and handled null and missing values
* Replaced invalid values (#N/A) with 0

5 DAX Calculations

* Calculated Average Delivery Days
* Created new tables for analysis
* Built Sales Forecast using order date and aggregated sales

6 Key Insights

* Customers prefer Cash on Delivery (COD)
* West region generates the highest sales
* Standard Class is the most used shipping mode
* Highest sales observed in Nov 2019 and Dec 2020
* Highest profit recorded in Mar 2020 and Dec 2019

7 Recommendations

* Promote COD payment options
* Focus on high-performing regions
* Optimize shipping strategies
* Plan inventory based on peak months
* Use forecasting for better demand planning
